Proyecto Cipher

Entre los diferentes cifrados que existen, uno de los más sencillos o vulnerables es el cifrado Cesar, por este motivo 
no diseñé una aplicación que sirva de cifrador o descifrador, ya que existen otros tipos de cifrado en línea que son más 
más seguros y gratuitos.

Si bien el cifrado Cesar es facilmente vulnerable en la práctica, se sigue usando como la introducción a la criptografía,
alumnos de diferentes carreras a nivel universitario o desarrolladores web deben comprenderlo como parte de sus objetivos
de aprendizaje, esta aplicación está dirigida a ellos.

Al ser una aplicación didactica, cuenta con 3 partes, (1)la interface de cifrado para que cuando ellos estén haciendo sus prácticas
tengan un cifrador con el que puedan comprobar sus resultados. (2) una sección donde se explica el algoritmo. (3) una sección con 
links a sitios de interes.

Las tres partes del producto se encuentran en una misma página web, pues cuando uno está haciendo cálculos e intentos es más fácil 
desplazarse de la teoría a la práctica si estos están juntos.
La aplicación brindará una manera de resolver el Cifrado Cesar y será una herramienta para los alumnos así como los demás links que
vendrán en ella como referencia al tema.
