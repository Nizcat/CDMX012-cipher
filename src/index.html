He encontrado varias fórmulas para desarrollar el Cifrado Cesar, la formula que se usó en esta aplicación para asignarle un valor ascii a cada letra y que sólo recorriendo un rango de letras regrese el valor adecuado, es:

  1. Fórmula para cifrar
  2. Variantes:
  3. rango (a...b)
  4. x=valor ascii asignado a una letra
  5. y=número que recorreremos
  6. r= residuo
  7. c= cociente
  8. s=suma
  9. Formula:
  10. paso1:
  11. s=x-a+y
  12. paso2:
  13. c= s/26
  14. r=c-s/26
  15. número cifrado =r+a
